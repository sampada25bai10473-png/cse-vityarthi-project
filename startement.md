1. Project Title:

To-Do List Manager

2. Overview of the Project:

A simple application that allows users to create, manage, and track their daily tasks.
It helps users stay organized by maintaining a structured list of pending and completed tasks.

3. Problem Statement:

People often forget daily tasks or struggle to manage them efficiently. This project aims
to solve that problem by providing a digital tool to store, update, and track tasks easily.

4. Scope of the Project:

Create, edit, and delete tasks

Mark tasks as completed

Categorize tasks

Maintain a clean, simple interface

Suitable for academic use and beginner-level development

5. Target Users:

Students

Working professionals

Anyone who wants to organize daily tasks efficiently

6. High-Level Features:

Add, delete, and update tasks

Mark task status as completed/pending

Display pending and completed tasks separately

Simple and user-friendly interface

7. Technologies / Tools Used:

Python

File handling (or optional database)

Tkinter (optional GUI)

8. Steps to Install and Run the Project:

Install Python 3

Download the project folder

Open Terminal/Command Prompt

Run the command:

python todo.py

9. Instructions for Testing:

Add sample tasks

Try editing and deleting them

Mark tasks as completed

Check if the completed task list updates correctly
